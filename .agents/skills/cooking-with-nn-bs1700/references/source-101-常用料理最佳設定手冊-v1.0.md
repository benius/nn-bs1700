# Panasonic NN-BS1700 常用料理最佳設定手冊 v1.0

## 使用原則（建議收藏）

| 食材 | 最佳模式 |
|------|----------|
| 冷凍肉類 | 微波低火 → 蒸氣 |
| 冷凍麵食 | 蒸氣 → 烘烤 |
| 生鮮肉類 | 烘烤 / 微波連動 |
| 青菜 | 蒸氣 |
| 冷藏便當 | 自動加熱 或 微波 |
| 魚類 | 蒸氣 |
| 厚肉排 | 微波連動 |
| 酥脆料理 | 烘烤或燒烤 |
