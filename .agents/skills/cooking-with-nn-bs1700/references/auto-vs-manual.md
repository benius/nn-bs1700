# 官方自動行程與手動最佳化對照

此檔內嵌整理自 `102-完全攻略_v1.0.docx`；具名料理設定仍須先套用 `source-priority.md`。

只在名稱、份量、容器、配件與層位都和機器畫面一致時使用自動行程。料理表中的 `AUTO` 只代表相應的自動解凍或完全相符的官方菜單入口；找不到完全相符項目就改手動，不猜號碼或相似行程。

## 目錄

- 常用自動入口
- 解凍、蔬菜與菜單搜尋
- 已核實編號與手動備援
- 自動與手動切換條件

| 官方入口／名稱 | 適用狀態 | 配件核心規則 | 手動備援 |

> 編號原則  本手冊不把「相似料理」硬套自動號碼。官方自動程序會依特定份量、容器、配件與放置位置設計，改材料後請切換手動。

## 自動與手動切換條件

| 條件 | 用自動 | 改手動 |
| --- | --- | --- |
| 料理名稱與機器一致 | 是 | 若份量／容器／配件不同 |
| 第一次做、配方照官方 | 是 | 想改厚度、糖油量或同時多盤 |
| 需要精準熟度 | 可先自動 | 最後用單面燒烤／短時微波補熱 |
| 冷凍市售食品 | 只有螢幕明載可冷凍直烹才用 | 其他照包裝或先解凍 |
| 連續烹調、爐內很熱 | 自動可能受U50限制 | 先跑爐內冷卻，再手動 |
| 食材屬高風險族群餐食 | 不把感測當唯一驗證 | 用食物溫度計確認 |
